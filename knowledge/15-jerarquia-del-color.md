# Jerarquia del color

El color no decora: dirige la mirada y jerarquiza la informacion. Esta es la guia que usa el especialista de color para que cada lead magnet tenga un foco claro y se vea de lujo. Cero emojis: el color se aplica a formas SVG, fondos, textos y estados.

## 1. Principio: el color es un sistema de prioridades
En una pantalla, lo que mas contrasta con su entorno es lo primero que se ve. Por eso el color se reparte por JERARQUIA, no por gusto:
- **Color de accion (1):** el mas saturado/contrastado, reservado para UNA cosa por pantalla (el CTA, el dato clave, el resultado). Si todo grita, nada grita.
- **Color de marca/acento (2):** refuerza identidad y guia secundaria (titulares, iconos activos).
- **Neutros (3):** fondos, texto de cuerpo, bordes. Son el 70-80 por ciento de la pantalla.
- **Estados:** exito, alerta, error, bloqueado, con significado consistente.

## 2. La regla 60-30-10
- 60 por ciento neutro dominante (fondo).
- 30 por ciento color secundario (superficies, tarjetas).
- 10 por ciento color de acento/accion (foco, CTA). Ese 10 por ciento es sagrado: solo para lo que quieres que toquen o lean primero.

## 3. Contraste = foco
- El ojo va al maximo contraste. Para crear foco, sube el contraste del elemento clave y baja el del entorno (oscurece o desatura lo de alrededor).
- Tecnica de "spotlight": fondo oscuro/desaturado + un unico punto luminoso (glow radial) sobre el elemento heroe.
- Evita el "confeti cromatico": muchos colores saturados compitiendo = cero jerarquia.

## 4. Temperatura y profundidad
- Calidos (rojo, ambar, durazno) avanzan y activan (urgencia, energia, accion).
- Frios (azul, malva, verde) calman y dan fondo/profundidad.
- Usa la temperatura para narrar fases (frio -> calor segun progresa el lead magnet o la escena).
- Profundidad: capas mas oscuras al fondo, mas claras/luminosas al frente.

## 5. Gradientes con criterio
- Gradientes de marca (2-3 paradas) para titulares heroe, CTAs y glows. No para texto de cuerpo.
- Mantener la familia cromatica; nada de arcoiris.

## 6. Accesibilidad (no negociable)
- Texto de cuerpo: contraste minimo 4.5:1; titulares grandes 3:1 (WCAG AA).
- Nunca transmitir estado SOLO con color (anadir icono SVG, texto o forma).
- Probar en modo claro/oscuro si aplica, y con simulacion de daltonismo.

## 7. Derivar la paleta del discovery
- Parte de los colores de marca del usuario (Gate 0). Si no hay, propon una paleta coherente con el nicho y la emocion (salud/sereno, finanzas/confianza, fitness/energia).
- Define tokens: --bg, --superficie, --texto, --muted, --acento, --accion, --exito, --alerta. Reusa en todo el lead magnet.

## 8. Checklist de jerarquia de color por pantalla
- [ ] Hay UN solo elemento con el color de accion.
- [ ] El fondo desatura/oscurece para que el heroe destaque.
- [ ] 60-30-10 respetado.
- [ ] Contraste AA en textos.
- [ ] Estados con color + forma/icono, no solo color.
- [ ] La temperatura apoya la narrativa.
